## 🚀 使用场景

### 场景 1：单聊
```
你: "贾国勋，帮我规划下个项目"
贾国勋: "收到！让我调用贾国翔..."
```

### 场景 2：群聊
```
你: "@贾国产 帮我写个 PRD"
贾国产: "好的，我来处理产品需求..."
```

### 场景 3：子任务
```
贾国勋: "测试任务交给贾国策..."
贾国策: "测试计划已完成..."
```

## 🎯 协作模式

### 模式 1：主 Agent 协调
```
用户 → 贾国勋（主 Agent）
         ↓
    判断任务类型
         ↓
    ┌──────────┬──────────┐
    ↓          ↓          ↓
贾国产    贾国炎    贾国策...
```

### 模式 2：直接对话
```
用户 → 特定 Agent（如 @贾国产）
         ↓
    直接处理任务
         ↓
    返回结果
```
